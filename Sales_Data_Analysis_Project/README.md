
##  Analysis Performed
1. **Data Exploration**: Import and examination of structured sales data
2. **Data Cleaning**: Handling missing values, duplicates, and data type conversions
3. **Excel Analysis**: 
   - PivotTables for revenue by product and region
   - Monthly revenue trends
   - Data visualization with charts
4. **Python Analysis**:
   - Data cleaning with pandas
   - KPI calculations
   - Sales representative performance analysis
   - Regional sales analysis

##  Technologies Used
- **Microsoft Excel**: Data exploration and visualization
- **Python**: Data analysis with pandas
- **Jupyter Notebooks**: Interactive analysis environment
- **Git/GitHub**: Version control and collaboration
